How to Use Markdown

Enter Markdown text below. Here are some basic symbols:

  • # for headers (e.g., # Header 1)
  • ## for subheaders (e.g., ## Header 2)
  • ### for smaller headers (e.g., ### Header 3)
  • ** for bold text (e.g., **bold**)
  • * for italic text (e.g., *italic*)
  • ~~ for strikethrough (e.g., ~~strikethrough~~)
  • ^ for superscript (e.g., ^superscript^)
  • ~ for subscript (e.g., ~subscript~)
  • 1. for numbered lists (e.g., 1. Item 1)
  • * or - for bullet points (e.g., - Item)
  • [Link Text](URL) for links (e.g., [Google](https://www.google.com))
  • ![Image](URL) for images (e.g., ![Image](image.jpg))
